# Дизайн иконки приложения «Нұр v2»

**Дата:** 2026-04-21
**Проект:** Ибадат Трекер (reportdeepen)
**Файл:** `assets/icon/app_nur.png` (1024×1024)

## Цель

Заменить текущую иконку на более выразительную версию того же концепта «нұр» (свет), гармонирующую с тёмно-индиго темой приложения (#0F172A → #1E1B4B, акцент #4F46E5).

## Концепция

Тёплый золотой луч света пробивается по диагонали сквозь фиолетово-индиговую «ночь», рассыпая звёздные искры. Ассоциация: рассвет, духовное озарение, «свет среди тьмы». Палитра — классика исламского искусства (золото на глубоком индиго, как в персидских и османских рукописях).

## Визуальные элементы

### Холст
- 1024×1024 px
- Скруглённые углы радиусом 180 px (стандарт app icon, iOS/Android адаптируют сами)

### Фон (радиальный градиент)
- Центр (смещён к низу-слева, ~30% / 70%): `#1E1B4B` (глубокий индиго)
- Края: `#0F0A2E` (почти чёрный с фиолетовым подтоном)
- Лёгкая виньетка по углам для «глубины»

### Луч света
- Направление: диагональ из верхнего правого угла вниз-влево, угол ~35°
- Три слоя (composition layers) с разной прозрачностью и gaussian blur для эффекта bloom/glow:
  1. Внешний ореол — широкий, сильно размытый, золотой `#F5B82E` alpha 0.25
  2. Средний слой — `#FFD369` (тёплое золото) alpha 0.7, умеренный blur
  3. Ядро — чистый белый `#FFFFFF` → золото, минимальный blur
- Ширина луча в самом ярком месте: ~30% ширины иконки
- Плавное растворение к противоположному углу (α → 0)
- Точка входа (верхний правый угол): белый блик-«корона» с радиальным свечением

### Звёздная россыпь
Рассыпаны преимущественно вдоль диагонали луча, плотнее к центру, реже по краям. Три типа:

1. **Золотые искры** — 4-конечные sparkle-звёзды (крест с утончающимися лучами), размер 8–14 px, цвет `#FFD369`, количество ~5. Расположены внутри яркой зоны луча.
2. **Белые/голубые точки** — простые круги, размер 2–4 px, цвета `#FFFFFF` и `#C7D2FE`, количество ~25. По всей зоне луча и чуть за её пределами.
3. **Индиговые точки в тени** — цвет `#4F46E5` alpha 0.4, размер 2–3 px, количество ~15. В тёмных зонах, создают «объёмность ночного неба».

### Акцентная деталь
В точке максимальной яркости луча — крошечная 8-конечная исламская звезда (геометрический ромб + квадрат), диаметром ~40 px, цвет `#FFFFFF` с золотым glow. Отсылка к 8 категориям ибадата, но ненавязчивая — воспринимается как геометрический акцент.

## Технический план реализации

1. Создать SVG 1024×1024 со всеми слоями, градиентами, SVG-фильтрами (`feGaussianBlur`, `feComposite`) для эффекта свечения. Файл: `assets/icon/app_nur.svg`.
2. Конвертировать SVG → PNG 1024×1024 через одну из утилит: `rsvg-convert`, ImageMagick (`magick convert`), Inkscape CLI, или онлайн-конвертер. Команду подберём по тому, что установлено у пользователя.
3. Сохранить бэкап старой иконки в `assets/icon/app_nur_v1_backup.png`.
4. Перезаписать `assets/icon/app_nur.png` новым файлом.
5. Пользователь запускает `flutter pub run flutter_launcher_icons` для регенерации Android/iOS launcher icons.

## Критерии приёмки

- Иконка читаема и эффектна в размерах 48×48, 96×96, 192×192, 512×512 (визуальный тест).
- Цвета соответствуют заявленным hex-кодам.
- Луч имеет чётко выраженный bloom-эффект.
- Композиция не симметрична, есть динамика (диагональ).
- На тёмных и светлых обоях домашнего экрана иконка не теряется.
- SVG исходник сохранён в `assets/icon/` для будущего редактирования.

## Что вне скоупа

- Splash screen (отдельная задача, если понадобится).
- Adaptive icon для Android (foreground/background раздельно) — сейчас используется один PNG через `flutter_launcher_icons` без adaptive настройки; при необходимости настроим отдельно.
- Смена названия приложения, бренд-айдентика, Play Store/App Store ассеты.
